# todo-app
Basit JavaScript To-Do List uygulaması

Bu proje HTML, CSS ve JavaScript kullanılarak geliştirilmiş basit bir görev takip uygulamasıdır.

## Özellikler
- Görev ekleme
- Görevleri tamamlandı olarak işaretleme
- LocalStorage ile verilerin saklanması

## Kullanılan Teknolojiler
- HTML
- CSS
- JavaScript

> Öğrenme amaçlı geliştirilmiştir.
